# **App Name**: Manitas Listas

## Core Features:

- User Accounts: User registration and authentication for both handyman service providers and customers.
- Service Catalog: Display available handyman services with descriptions and pricing.
- Quotation Request: Allow users to request quotations for specific services.
- Service Management: Allow handymen to manage appointments, track progress, and send notifications.
- Handyman Directory: Enable users to browse through handyman profiles and ratings to find the right provider for their needs
- Customer service & Support: Enable users to browse through handyman profiles and ratings to find the right provider for their needs
- AI Solution Assistant: AI powered assistant that takes a description of a problem from the customer and suggests potential solutions based on handyman skills. The tool uses reasoning to decide when or if to include handyman skill information.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for trust and reliability in handyman services.
- Background color: Light cyan (#E0FFFF), a very light tint of the primary hue, creating a clean backdrop.
- Accent color: Teal (#008080), providing contrast and drawing attention to key interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) to offer a blend of modernity and warmth, readable for both titles and content.
- Use clear, simple icons to represent different service categories.
- A clean and intuitive layout that allows users to easily find and request services.
- Subtle animations when the app processes user requests to improve the user experience.
